# Team Task Manager

A full-stack Team Task Manager application built using Next.js, TypeScript, MongoDB, and Tailwind CSS.

## Features

- User Signup & Login
- JWT Authentication
- Role-based Access
- Create & Manage Projects
- Create & Manage Tasks
- Task Status Update
- Dashboard with Task Statistics
- MongoDB Integration
- REST APIs
- Railway Deployment

---

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- MongoDB
- Mongoose
- JWT
- Railway

---

## Installation

```bash
npm install
npm run dev
```
